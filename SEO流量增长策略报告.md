## 五、搜索引擎双引擎策略

### 5.1 百度 vs Google 策略差异

| 维度 | 百度策略 | Google策略 |
|------|---------|-----------|
| **备案** | 必须ICP备案，无备案严重降权 | 无硬性要求 |
| **生态** | 重点布局：站长平台/地图/知道/贴吧/文库 | 重点布局：GSC/GMB/Reviews |
| **内容** | 偏好中文原创+本地化信息 | 偏好E-E-A-T权威内容 |
| **结构化数据** | LocalBusiness/FAQPage最重要 | 同左+增加BreadcrumbList |
| **速度** | 国内服务器是前提 | Cloudflare Pages可以接受 |
| **移动端** | 移动优先索引已全面启用 | 同左 |
| **社交信号** | 微信生态有一定影响 | 不直接参考社交信号 |

### 5.2 当前状态建议

当前网站部署在 **Cloudflare Pages**（海外服务器），对Google相对友好但对百度不利。

**推荐的双轨策略**：
1. **短期（当前）**：保持现有站点不变，重点做好内容和技术优化，Google先跑起来
2. **中期（1-2个月内）**：申请国内服务器 + ICP 备案 + .cn域名，将主站迁回国内，彻底解决百度的服务器问题

## 八、风险提示与持续优化

### 8.1 需要关注的风险点

1. **服务器位置**：Cloudflare Pages 在中国大陆访问慢 → 尽快迁移
2. **域名年龄**：域名较新，初始信任度低 → 靠时间积累
3. **内容更新频率**：目前为静态站，缺乏定期更新机制 → 建议后续加博客模块
4. **外部链接**：目前外链几乎为零 → 需要持续推进外链建设
5. **用户互动**：缺少评价/评论系统 → 百度地图评价可部分弥补

### 8.2 持续优化检查清单

- [ ] **每周**：检查 Search Console / 百度站长平台的抓取错误和索引状态
- [ ] **每两周**：监控目标关键词排名变化（Top 3/10/20 变化）
- [ ] **每月**：分析哪些页面带来了最多流量，是否需要进一步优化
- [ ] **每季度**：审查并更新过时的信息（价格、联系方式等）
- [ ] **每半年**：全面审计一次SEO健康状况，对标竞争对手调整策略
